Welcome to LuciadLightspeed and LuciadFusion's OGC web services

LuciadLightspeed and LuciadFusion's OGC web services include OGC-certified WMS, WFS-T, WCS, CSW server components, enabling users to exchange maps, vector features and raster coverages in a standardized and interoperable way. This server sample includes a ready-to-use WMS, WFS-T, WCS and CSW, configured with sample data sources available in LuciadLightspeed and LuciadFusion. Through the API, each of the services can be fully customized to meet user requirements.

Note that CSW is only available in LuciadFusion.

To find out more information about a service together with a few example requests and responses, click on one of the menu items.

To see the services in action, start one of the available client application samples:

  • OGC WMS:
    • 2D view: wms.client.gxy or wms.client.sld
    • 2D/3D hardware-accelerated view: wms.client.lightspeed
  • OGC WFS-T:
    • 2D view: ogc.wfs.client.gxy
    • 2D/3D hardware-accelerated view: ogc.wfs.client.lightspeed
  • OGC WCS:
    • 2D view: ogc.wcs.client
